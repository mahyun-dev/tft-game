# 챔피언 추천 조합 메타 가이드

이 가이드는 현재 메타에서 강력한 챔피언 조합과 시너지 전략을 설명합니다. 각 조합은 부대별 시너지와 역할 분배를 고려하여 구성되었습니다.

## 메타 조합 개요

### 현재 메타 트렌드
- **경수색반 중심**: 기동성과 화력의 균형
- **중수색반 + 포병**: 중화력과 범위 공격
- **공병 + 의무**: 폭발 피해와 생존
- **특수전**: 암살과 기습 전략

## 추천 조합 1: 경수색반 기동타격대

### 조합 구성 (8명)
- **코어 유닛**:
  - 운전병 x2 (1코스트, 암살자)
  - 척후병 x2 (2코스트, 암살자)
  - 저격수 x1 (4코스트, 암살자)
  - 포수 x1 (2코스트, 딜러)
  - 중화기병 x1 (3코스트, 딜러)
  - 경수색반장 x1 (2코스트, 탱커)

### 시너지 분석
- **경수색반 (6)**: 이동속도 +50%, 공격속도 +50%, 회피 +35%, 첫 공격 2배
- **보병 (4)**: 방어력 +20, 체력 +150
- **기동 (3)**: 이동속도 +25%, 회피 +15%

### 아이템 전략
- **암살자**: 은신화 + 저격소총
- **딜러**: 속사 K-2 + 폭발탄
- **탱커**: 완전군장 + 구급낭

### 장점
- 최고의 기동성으로 상대를 압도
- 암살자들의 기습 공격
- 탱커의 생존으로 후반 유지

## 추천 조합 2: 중수색반 포병중심

### 조합 구성 (8명)
- **코어 유닛**:
  - 통신병 x2 (1코스트, 서포터)
  - K-1 포수 x2 (4코스트, 탱커)
  - 대대장 x1 (5코스트, 서포터)
  - 공병 x2 (3코스트, 서포터)
  - 중대장 x1 (4코스트, 서포터)

### 시너지 분석
- **중수색반 (4)**: 방어력 +40, 체력 +300, 반사 피해 +30%
- **포병 (4)**: 스킬 피해 +40%, 마나 +30
- **공병 (3)**: 체력 +200, 방어력 +30, 스킬 범위 +50%

### 아이템 전략
- **딜러**: 저격소총 + 중기관총
- **탱커**: 중장갑 + 첨단 헬멧
- **서포터**: C-4 폭약 + 야전병원

### 장점
- 강력한 방어력과 생존
- 포병의 범위 스킬 피해
- 공병의 폭발 지원

## 추천 조합 3: 특수전 암살단

### 조합 구성 (8명)
- **코어 유닛**:
  - 운전병 x2 (1코스트, 암살자)
  - 척후병 x2 (2코스트, 암살자)
  - 저격수 x2 (4코스트, 암살자)
  - 중화기병 x1 (3코스트, 딜러)
  - 포수 x1 (2코스트, 딜러)

### 시너지 분석
- **특수전 (5)**: 공격력 +50, 치명타 확률 +25%, 은신 3초
- **경수색반 (4)**: 이동속도 +30%, 공격속도 +30%, 회피 +20%

### 아이템 전략
- **암살자**: 은신화 + 저격소총
- **딜러**: 저격소총 + 속사 K-2
- **서포터**: 야전병원 (필요시)

### 장점
- 은신과 기습의 극치
- 높은 치명타 피해
- 상대의 카운터 플레이 무력화

## 추천 조합 4: 의무+공병 지원대

### 조합 구성 (8명)
- **코어 유닛**:
  - 의무병 x2 (1코스트, 서포터)
  - 정비병 x2 (2코스트, 서포터)
  - 공병 x2 (3코스트, 서포터)
  - 중대장 x1 (4코스트, 서포터)
  - 중화기병 x1 (3코스트, 딜러)

### 시너지 분석
- **의무 (5)**: 체력 +400, 초당 체력 회복 +50, 아군 치유 +20%
- **공병 (4)**: 체력 +300, 방어력 +40, 스킬 범위 +50%, 광역 피해 +40%
- **보병 (3)**: 방어력 +20, 체력 +150

### 아이템 전략
- **서포터**: 고급 구급낭 + C-4 폭약
- **탱커**: 완전군장 + 고급 구급낭
- **딜러**: 속사 K-2 + 폭발탄

### 장점
- 뛰어난 생존과 회복
- 강력한 스킬 지원
- 후반까지 유지력 우수

## 조합별 카운터 전략

### 경수색반 기동타격대 카운터
- **상대 조합**: 중수색반 포병중심
- **카운터 이유**: 중수색반의 높은 방어력과 반사 피해
- **대응**: 공병 추가로 폭발 피해 극대화

### 중수색반 포병중심 카운터
- **상대 조합**: 특수전 암살단
- **카운터 이유**: 암살자들의 기습과 은신
- **대응**: 의무병 추가로 생존 강화

### 특수전 암살단 카운터
- **상대 조합**: 의무+공병 지원대
- **카운터 이유**: 강력한 회복과 생존
- **대응**: 저격수 추가로 원거리 화력

### 의무+공병 지원대 카운터
- **상대 조합**: 경수색반 기동타격대
- **카운터 이유**: 기동성과 암살자들의 기습
- **대응**: 중수색반 추가로 방어력 강화

## 라운드별 전략

### 1-3라운드
- 1코스트 유닛으로 시작
- 생존 아이템(방탄복, 헬멧) 우선
- 시너지 코어 구축

### 4-6라운드
- 2-3코스트 유닛 추가
- 역할별 아이템 분배
- 시너지 티어 업그레이드

### 7-9라운드
- 고코스트 유닛 배치
- 최종 시너지 완성
- 아이템 최적화

## 팁과 전략

1. **시너지 우선**: 시너지 티어를 먼저 맞추고 아이템은 나중에
2. **롤 분배**: 탱커 2-3명, 딜러 3-4명, 서포터 2-3명 유지
3. **아이템 타이밍**: 4라운드부터 아이템 조합 시작
4. **카운터 플레이**: 상대 조합에 맞는 카운터 유닛 준비
5. **경제 관리**: 아이템 리롤과 경험치 구매 적절히 사용

## 자주 하는 실수

- **시너지 무시**: 아이템만 쫓다가 시너지 실패
- **롤 불균형**: 딜러만 많아 생존 실패
- **아이템 낭비**: 필요 없는 아이템에 골드 소모
- **타이밍 실수**: 고코스트 유닛을 너무 일찍 배치
