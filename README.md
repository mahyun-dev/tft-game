# 11사단 기갑수색대대 전술 체스

🪖 HTML + JavaScript로 만든 완전한 전략 오토배틀러 게임  
🤖 8명 배틀로얄: 플레이어 1명 + AI 7명  
💡 3단계 AI 난이도 (쉬움 / 보통 / 어려움)  
✨ 30초 전략 타이머 + 자동 전투 시스템  
🎯 드래그 앤 드롭 아이템 장착, 챔피언 툴팁  

## 실행 방법
1. `index.html` 파일을 브라우저에서 열기
2. [혹은 여기를 클릭하여 실행하기](https://mahyun-dev.github.io/tft-game/index.html)

## 조작법
- **마우스 드래그**: 챔피언을 벤치 ↔ 필드로 이동
- **아이템 드래그**: 보관함에서 챔피언에게 장착 (최대 3개)
- **챔피언 호버**: 자세한 정보 툴팁 표시
- **상점**: 챔피언 구매 (클릭) 및 리롤 (2G)
- **경험치 구매**: 4골드로 4 경험치 구매
- **상대 필드 보기**: 다른 플레이어들의 필드와 시너지 확인

## 게임 시스템

### 🎮 배틀로얄 시스템
- **8명 플레이어**: 당신 + AI 7명
- **랜덤 매칭**: 매 라운드마다 다른 상대와 전투
- **AI 난이도 분포**: 쉬움 1명, 보통 4명, 어려움 2명
- **체력 시스템**: 전투에서 패배하면 체력 감소
- **순위 시스템**: 체력 0이 되면 탈락, 마지막 생존자가 1등!
- **AI vs AI 전투**: 당신이 대전하지 않는 AI들끼리도 싸움

### ⏱️ 라운드 시스템
- **30초 준비 시간**: 챔피언 배치, 아이템 장착, 레벨업
- **자동 전투 시작**: 시간 종료 시 자동으로 전투 돌입
- **PVE 라운드**: 1, 2, 3, 9, 18, 27 라운드는 크립 라운드 (아이템 보상)
- **PVP 라운드**: 나머지 라운드는 다른 플레이어와 전투

### 💰 골드 시스템
- **기본 수입**: 매 라운드 5골드
- **이자**: 10골드당 1골드 (최대 5골드)
- **연승/연패 보너스**: 전투 결과에 따른 추가 골드
- **용도**: 챔피언 구매, 상점 리롤, 경험치 구매

### ⭐ 레벨 시스템
- **자동 경험치**: 매 라운드 2 경험치 자동 획득
- **경험치 구매**: 4골드로 4 경험치 구매 가능
- **필드 크기**: 레벨에 따라 배치 가능한 챔피언 수 증가
- **상점 확률**: 레벨이 높을수록 고급 챔피언 확률 증가
